Introduction

About

Roosa is a HTML admin or dashboard template powered with Twitter Bootstrap 3.3.7 framework & HTML5 Boilerplate, both are the perfect combination to start in making a website template. Roosa can be used for any type of web applications like Blogging Dashboard, Monitoring Dashboard, Shop Cak End etc. Roosa has a modern flat design which makes your next project look awesome and yet user-friendly. Roosa has 20+ collection of jQuery plugins and UI components and support on all major web browsers, tablets and phones.Roosa comes with 4 layout variations. Once you purchased Roosa, you will be entitled to free download of all future updates for the same license.

Here are some key features of Roosa :


  • Bootstrap 3.x.x based
  • HTML5 Boilerplate
  • 4 Layout Variations
  • Responsive Design
  • Build with LESS
  • 20+ jQuery Plugins
  • 2000+ Font Icons
  • 10+ Ready Pages
  • Fixed & Static Layout

Getting Started

Preparation

Make sure you've set up multiple application support for developing and further editing the template. All you need to do so is a code editor like Sublime, Notepad ++, Atom or other code editor application according to your taste. In this case, we use the Sublime when doing development. Then, of course, you also have to have a Web browser that supports the latest web technologies like CSS3 and HTML5.

  1. Sublime
  2. Notepad++
  3. Atom
  4. Google Chrome
  5. XAMPP

Installation

Installing HTML template is very easy, you do not need to do any configuration. The easiest way is to access via file:// or by double clicking on the HTML file. But sometimes the way it will make the most of the jQuery plugins not running properly, then it is good to have a local server application on your computer such as XAMPP. What you should do after installing a local server application is to copy all of the assets templates into the htdocs folder and access them through a web browser such as Google Chrome.

Basic Template

Start with this basic HTML template, or modify the starter pages. We hope you'll customize our templates and starter pages, adapting them to suit your needs.

Copy the HTML below to begin working with a minimal HTML document.

<!DOCTYPE html>
<html lang=en>

<head>
	<meta charset="utf-8">
	<meta http-equiv="X-UA-Compatible" content="IE=edge">
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<meta name="description" content="">
	<meta name="author" content="">
	<title>Roosa - Just Another Dashboard Template</title>

	<!-- Fonts -->
	<link href="https://fonts.googleapis.com/css?family=Roboto:300,400,500,700" rel="stylesheet">

	<!-- Framework -->
	<link rel="stylesheet" href="dist/css/vendor/bootstrap.min.css">

	<!-- Main Template CSS -->
	<link rel="stylesheet" href="dist/css/bootstrap.custom.css">
	<link rel="stylesheet" href="dist/css/style.css">
	<link rel="stylesheet" href="dist/css/helper.css">
</head>


<body>

	<div id="rs-wrapper">

		<h1>Hello World</h1>

	</div><!-- /#rs-wrapper -->

	
	<script src="dist/js/vendor/modernizr.min.js"></script>
	<script src="dist/js/vendor/jquery.min.js"></script>
	<script src="dist/js/vendor/bootstrap.min.js"></script>

	<!-- Template Js -->
	<script src="dist/js/apps.js"></script>

</body>
</html>

Creating First Page

Roosa has four layout variations, and each variation has four types of positions. Such a static and fixed position.

Use the tags that you've copied from a basic template, and make sure all the appropriate assets in place. Then we started on the inside of the <div id="rs-wrapper">, ie remove the text Hello World.

Copy the HTML below insdie the wrapper tag.

<!-- BEGIN MAIN HEADER NAV -->
<nav class="navbar navbar-default rs-navbar navbar-static-top">
	<div class="container-fluid">

		<!-- Begin navbar header -->
		<div class="navbar-header has-right-divider">

			<!-- Begin Logo Brand -->
			<div class="rs-logo fixed-width">
				<a class="navbar-brand text-uppercase rs-brand-text" href="">Roosa</a>
			</div><!-- /.rs-logo -->
			<!-- End Logo Brand -->

			<!-- Begin button to toggle sidebar -->
			<button type="button" class="navbar-toggle collapsed sidebar-toggle" id="rs-sidebar-toggle-mobile">
				<span class="icon-bar"></span>
				<span class="icon-bar"></span>
				<span class="icon-bar"></span>
			</button>
			<!-- End button to toggle sidebar -->

		</div><!-- /.navbar-header -->
		<!-- End Navbar header -->

	</div><!-- /.container-fluid -->
</nav>
<!-- END MAIN HEADER NAV -->


<!-- BEGIN SIDEBAR NAV -->
<aside class="rs-sidebar">
	Sidebar
</aside><!-- /.rs-sidebar -->
<!-- END SIDEBAR NAV -->


<!-- BEGIN MAIN CONTENT -->
<article class="rs-content-wrapper">
	<div class="rs-content">
		<div class="rs-inner">
			<!-- Begin default content width -->
			<div class="container-fluid">

				Content

			</div><!-- /.container-fluid -->
		</div><!-- /.rs-inner -->
	</div><!-- /.rs-content -->
</article><!-- /.rs-content-wrapper -->
<!-- END MAIN CONTENT -->


<!-- BEGIN FOOTER -->
<footer class="rs-footer absolute-footer small">
	Footer
</footer>
<!-- END FOOTER -->

Page Layouts

Static Layout

All you need to do to make the static page is let the <div id="rs-wrapper"> like this without adding any other classes, then use .navbar-static-top for the nav

Template Demo

check the HTML below :

<div id="rs-wrapper">

	<nav class="navbar navbar-default rs-navbar navbar-static-top">
	
	...

Fixed Nav Layout

Add .rs-header-fixed-top into <div id="rs-wrapper"> then use .navbar-fixed-top for the nav

Template Demo

check the HTML below :

<div id="rs-wrapper" class="rs-header-fixed-top">

	<nav class="navbar navbar-default rs-navbar navbar-fixed-top">
	
	...

Fixed Layout

It means the header nav & sidebar has fixed position. Please note that we can't make fixed position for sidebar only, it will break the template UX and UI.

Make sure <div id="rs-wrapper"> has .rs-header-fixed-top and the nav has .navbar-fixed-top. Then add this class .rs-sidebar-fixed for wrapper. Also if we want to use custom scroll bar on sidebar, then we should add .rs-scroll-custom into .rs-sidebar

Template Demo

check the HTML below :

<div id="rs-wrapper" class="rs-header-fixed-top rs-sidebar-fixed">

	<nav class="navbar navbar-default rs-navbar navbar-fixed-top">
	
	<aside class="rs-sidebar rs-scroll-custom">

Boxed Layout

Please note that the boxed layout only support for static layout and fixed nav layout.

Add .rs-boxed-layout into body and also add color class helper to make body background color, for example we use gradient color like .bg-grad-03. See more gradent color classes here

Template Demo

check the HTML below :

<body class="rs-boxed-layout bg-grad-03">

Layout Variations

Default Sidebar Layout

The example above are using default sidebar layout, you only need to call Javascript file into your HTML page to use this layout variation.

Template Demo

<!-- Additional JS File -->
<script src="dist/js/layout-default.js"></script>

Mini Sidebar Layout

The second variation is mini sidebar layout, this layout presented small sidebar size with iconic menu. The text menu and sub menu will show when we hover one of the icons.

The mini sidebar only works on desktop devices with minimum width is 768px, under that width or opened in mobile device (such as Ipad, Tablets or Phones) in any resolutions, the sidebar automatically change to default size. Check the live demo with your mobile devices and the desktop too see the difference.

Template Demo

<!-- Additional CSS File -->
<link rel="stylesheet" href="dist/css/sidebar-mini.css">

<!-- Additional JS File -->
<script src="dist/js/layout-mini-sidebar.js"></script>

Iconic Sidebar Layout

The third variation is iconic sidebar layout, this layout presented small sidebar size with iconic menu. The text menu show as tooltips, and it's not support multilevel menu. It means this variation presented simplicity template for simple web app.

The sidebar nav will move to bottom on device under 738px, check this template on your mobile devices.

Template Demo

<!-- Additional CSS File -->
<link rel="stylesheet" href="dist/css/sidebar-iconic.css">

<!-- Additional JS File -->
<script src="dist/js/layout-icon-sidebar.js"></script>

Horizontal Menu Layout

The last variation is horizontal menu layout, this layout converts default sidebar into horizontal top menu.

The horizontal menu only works on desktop devices with minimum width is 1200px, under that width or opened in mobile device (such as Ipad, Tablets or Phones) in any resolutions, the sidebar automatically change to default size. Check the live demo with your mobile devices and the desktop too see the difference.

Template Demo

<!-- Additional CSS File -->
<link rel="stylesheet" href="dist/css/horizontal-menu.css">

<!-- Additional JS File -->
